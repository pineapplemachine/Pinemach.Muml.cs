MuDocument

Represents a Muml document, typically corresponding to a single file. A document is a collection of one or more top-level Muml elements, plus optional header content.

MuDocumentContentComparer

Comparer which checks equality of document content.

Represents an element within a Muml document. Elements are the fundamental unit of structure and data within a document.
